Led the frontend architecture and delivery of a secure Debt Management Portal used by third-party debt collection partners to manage accounts and transactions within a regulated financial environment. Designed a scalable React and TypeScript application, collaborated with backend, platform and security teams to define robust API contracts and integrate enterprise Single Sign-On (SSO). Introduced a shared component library and frontend architecture standards that improved consistency, reduced technical debt and delivered a secure, maintainable platform for long-term growth.
